# 🎬 Netflix Data Analysis

## 📌 Project Overview

This project analyzes the Netflix Movies and TV Shows dataset to uncover insights about content trends, production, and distribution over time.

The goal is to understand how Netflix content has evolved and what factors dominate the platform.

---

## 📂 Dataset

* Source: Netflix Movies and TV Shows Dataset (Kaggle)
* Features include:

  * Title
  * Type (Movie/TV Show)
  * Director
  * Cast
  * Country
  * Release Year
  * Date Added
  * Genre

---

## 🛠️ Tools & Technologies Used

* Python 🐍
* Pandas 📊
* Matplotlib 📈
* Seaborn 🎨

---

## 🔍 Analysis Performed

### 1. Movies vs TV Shows

* Compared the number of movies and TV shows available on Netflix

### 2. Top Producing Countries

* Identified which countries contribute the most content

### 3. Year-wise Content Addition

* Analyzed how content addition has grown over time

### 4. Genre Distribution

* Studied the most common genres on Netflix

---

## 📊 Key Insights

* 🎥 Movies dominate Netflix content compared to TV Shows
* 🌍 United States produces the highest number of titles
* 📈 Massive growth in content after 2016
* 🎭 Drama and International genres are highly popular
