# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows dataset to extract insights about content trends.

## Dataset
Netflix Movies and TV Shows Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Movies vs TV Shows comparison
- Top producing countries
- Year-wise content addition
- Genre distribution

## Key Insights
- Movies dominate Netflix content
- United States produces the most titles
- Netflix content increased rapidly after 2016
